Snake Game 

This is a classic Snake game built using Java and Swing.
The player controls the snake with the arrow keys, eats apples to grow, and tries to avoid hitting the walls or itself.

Features

- Smooth movement with keyboard controls

- Score tracking

-Random apple generation

-Game Over screen

Controls

 Left Arrow, Right Arrow, Up Arrow, Down Arrow

This project was made for practice with Java, Swing, and basic game logic.
